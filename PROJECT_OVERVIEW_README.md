# Project Overview: Real-Time Stock Market Dashboard

## Objective
The primary objective of this project is to create a real-time stock market dashboard.

## Key Phases & Tasks
1. **Data Acquisition:** Fetching financial data from external sources (e.g., Yahoo Finance API, Stocktwit API).
2. **Exploratory Data Analysis (EDA):** Understanding features through statistical methods like moving averages, hypothesis tests, and confidence intervals.
3. **Data Visualization:** Creating box plots, histograms, and density plots to illustrate stock spread, moving averages, volatility, and P/E ratios.
4. **Web Dashboard Integration:** Building and deploying the base web application to showcase insights using Streamlit and Heroku.
5. **Machine Learning Pipeline:**
   - Problem formulation.
   - Preprocessing (handling volatility, outliers, missing data, imbalance, and scaling).
   - Model selection, setting benchmarks, and feature engineering.
   - Hyperparameter tuning and experimenting with ensembles (Random Forest, Gradient Boosting Decision Trees).
   - Validation and finalizing the model.
   - Predicting on test and real-time data, followed by performance checks.
6. **Deployment & Scaling:** Deploying a fully functional prototype, then scaling up by integrating sequence models (Deep Learning/LSTMs) and natural language inference from web-scraped financial news.
