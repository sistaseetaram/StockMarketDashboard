# Project Summary

## Objective
The main objective of this project is to create a real-time stock market dashboard.

## Task Phases
1. **Data Acquisition**: Fetching data from sources like the Yahoo Finance API and Stocktwit API.
2. **Exploratory Data Analysis (EDA)**: Understanding features and data via statistical methods, such as moving averages, hypothesis testing, and confidence intervals.
3. **Data Visualization**: Creating visual representations, including box plots for price spreads, histograms/density plots for volatility/moving averages, and density plots for features like price-to-earnings ratios.
4. **Web Dashboard**: Building the base web application using Streamlit to showcase insights and deploying it to Heroku.
5. **Machine Learning Problem Formulation**: Identifying specific tasks or problems solvable using ML.
6. **Preprocessing**: Handling data volatility, outliers, missing data, class imbalance, and scaling.
7. **Model Selection and Feature Engineering**: Establishing benchmark linear models, analyzing feature importance, engineering new features, and selecting the final feature set.
8. **Hyperparameter Tuning and Validation**: Experimenting with different models, tuning hyperparameters, using ensemble techniques (RF, GBDT), and finalizing the model.
9. **Prediction and Performance Check**: Predicting outcomes on test and real-time data, and evaluating performance with appropriate metrics.
10. **Prototype Deployment**: Deploying the fully functioning ML prototype.
11. **Scaling Up**: Adding advanced functionalities like deep learning sequence models and Natural Language Inference on stock news using web scraping and LSTMs.
