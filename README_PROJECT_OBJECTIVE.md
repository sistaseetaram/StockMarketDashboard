# Project Objective and Summary

## Objective
The primary objective of the **StockMarketDashboard** project is to create a real-time stock market dashboard.

## Key Phases and Tasks
1. **Data Acquisition**: Fetch financial data using APIs (e.g., Yahoo Finance, Stocktwit).
2. **Exploratory Data Analysis (EDA)**: Understand data through statistical methods like moving averages, hypothesis testing, and confidence intervals.
3. **Data Visualization**: Visualize spread, moving averages, volatility, and features using box plots, histograms, and density plots.
4. **Web Dashboard**: Develop a foundational web dashboard using Streamlit to present insights and deploy it on Heroku.
5. **Machine Learning Formulation & Preprocessing**: Define ML tasks. Process data to handle volatility, outliers, missing values, imbalances, and scaling.
6. **Modeling**: Select benchmark linear models, engineer features, and move on to ensemble models (Random Forest, GBDT). Tune hyperparameters and validate models.
7. **Prediction & Performance**: Run predictions on test and real-time data and evaluate performance.
8. **Deployment & Scaling**: Deploy the functioning prototype and scale up the project with deep learning (sequence models) and NLP for stock news inference.
