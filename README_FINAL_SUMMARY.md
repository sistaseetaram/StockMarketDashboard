# Project Summary: Stock Market Dashboard

## Objective
The primary objective of this project is to create a real-time stock market dashboard.

## Key Tasks & Development Stages
The project is structured around several core tasks:

1. **Data Acquisition:**
   - Fetching financial data using the Yahoo Finance API and Stocktwit API.

2. **Exploratory Data Analysis (EDA):**
   - Analyzing features and data via statistical methods (moving averages, hypothesis testing, confidence intervals).

3. **Data Visualization:**
   - Creating box plots for opening/closing prices.
   - Using histograms and density plots to illustrate moving averages and volatility.
   - Visualizing key financial metrics like Price-to-Earnings (P/E) ratio.

4. **Web Dashboard Development:**
   - Building the primary web application with Streamlit to display insights and visualizations, and deploying it to Heroku.

5. **Machine Learning Pipeline:**
   - **Problem Formulation:** Defining specific tasks to solve using Machine Learning.
   - **Preprocessing:** Handling data volatility, outliers, missing data, and imbalances, followed by data scaling.
   - **Model Selection & Feature Engineering:** Establishing benchmarks with linear models, adding features iteratively, and selecting the final feature set.
   - **Hyperparameter Tuning:** Experimenting with various models and ensembles (Bagging, Boosting, Stacking like Random Forest and GBDT), and finalizing the model.
   - **Prediction & Performance Check:** Running predictions on test and real-time data, and evaluating performance.

6. **Deployment & Scaling:**
   - Deploying a fully functional prototype.
   - Scaling up the project with advanced sequence models (Deep Learning) and Natural Language Inference on stock news using web scraping and LSTMs.
