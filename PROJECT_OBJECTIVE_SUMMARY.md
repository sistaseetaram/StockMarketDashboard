# Project Objective Summary

**Objective:** Create a real-time stock market dashboard.

## Summary of Tasks to Reach the Objective:
1. **Data Acquisition:** Gather data from sources like Yahoo Finance API and Stocktwit API.
2. **Exploratory Data Analysis (EDA):** Understand features and data using statistical methods (e.g., moving averages, hypothesis testing, confidence intervals).
3. **Data Visualization:** Create box plots, histograms, and density plots to show stock price spread, volatility, and features like P/E ratio.
4. **Initial Web Dashboard:** Build a base web application to display insights using Streamlit and deploy it to Heroku.
5. **Machine Learning Formulation:** Define the specific tasks or problems to solve using Machine Learning.
6. **Data Preprocessing:** Handle data volatility, outliers, missing data, and imbalances, as well as scale the data.
7. **Model Selection & Feature Engineering:** Set benchmarks with simple linear models, analyze feature importance, and engineer new features.
8. **Hyperparameter Tuning & Validation:** Experiment with different models and ensemble techniques (Bagging, Boosting, Stacking like RF, GBDT), tune hyperparameters, and finalize the model.
9. **Prediction & Performance Check:** Make predictions on test and real-time data and evaluate performance using appropriate metrics.
10. **Prototype Deployment:** Deploy the fully functioning prototype model.
11. **Scaling Up:** Expand the project by developing Deep Learning sequence models and applying Natural Language Processing (NLP/LSTM) on web-scraped stock news.
