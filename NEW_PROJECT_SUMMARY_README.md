# Stock Market Dashboard Project Summary

## Objective
The primary objective of this project is to create a real-time stock market dashboard.

## Summary
The dashboard serves as an interactive web application that provides users with insights into stock market data, covering everything from basic data acquisition and exploratory data analysis (EDA) to advanced machine learning predictions.

The project outlines a comprehensive path:
1. **Data Acquisition:** Gathering data from sources such as Yahoo Finance API and Stocktwit API.
2. **EDA & Visualization:** Utilizing statistical methods and visualizations (box plots, histograms, density plots) to analyze and understand stock behavior, volatility, and key financial ratios.
3. **Dashboard Creation:** Developing the core web dashboard using Streamlit to visualize the derived insights and deploying it to Heroku.
4. **Machine Learning Pipeline:** Formulating problems to solve with ML, followed by preprocessing (handling volatility, outliers, imbalances), model selection, feature engineering, and hyperparameter tuning. The pipeline tests ensembles and benchmarks linear models.
5. **Prediction & Deployment:** Generating predictions on test and real-time data, checking performance, and deploying the prototype model.
6. **Future Scalability:** Enhancing the platform with Deep Learning sequence models (like LSTMs) and incorporating Natural Language Inference to parse stock news via web scraping for more comprehensive analyses.
