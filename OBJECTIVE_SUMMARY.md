# Project Objective Summary

The main objective of this project is to create a **real-time stock market dashboard**.

## Task Summary
The project follows a comprehensive pipeline to build the dashboard and eventual predictive models. The key tasks include:

1.  **Data Acquisition**: Sourcing financial data from Yahoo Finance and Stocktwit APIs.
2.  **Exploratory Data Analysis (EDA)**: Utilizing statistical methods to understand features, moving averages, and volatility.
3.  **Data Visualization**: Implementing box plots, histograms, and density plots to illustrate stock price spreads and volatility.
4.  **Web Dashboard Creation**: Building a base web application using Streamlit to showcase insights and deploying it to Heroku.
5.  **Machine Learning Setup**: Formulating ML problems, preprocessing data (handling volatility, outliers, missing data, and scaling), and performing feature engineering.
6.  **Modeling & Tuning**: Selecting benchmark models, checking feature importance, experimenting with ensembles (Bagging, Boosting, Stacking), and tuning hyperparameters to finalize the model.
7.  **Prediction & Deployment**: Testing predictions on real-time data, checking performance with metrics, and deploying the prototype model.
8.  **Scaling Up**: Adding sequence/Deep Learning models and implementing Natural Language Inference from web-scraped stock news using LSTMs.
