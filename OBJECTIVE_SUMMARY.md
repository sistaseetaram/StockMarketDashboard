# Project Objective Summary

## Objective
The primary objective of this project is to create a real-time stock market dashboard.

## Key Tasks
- **Data Acquisition**: Fetch data from sources like Yahoo Finance API and Stocktwit API.
- **Exploratory Data Analysis (EDA)**: Understand features and data using statistical methods (e.g., moving average, hypothesis tests, confidence intervals).
- **Data Visualization**: Create visual representations such as box plots for price spread, histograms/density plots for volatility/moving averages, and density plots for P/E ratios.
- **Web Dashboard**: Build the initial web application using Streamlit to showcase insights, and deploy it to Heroku.
- **Machine Learning Problem Formulation**: Define ML-solvable problems or tasks.
- **Preprocessing**: Handle data volatility, outliers, missing data, imbalance, and scaling.
- **Model Selection & Feature Engineering**: Establish benchmark linear models, evaluate feature importance, add/experiment with new features, and determine the final feature set.
- **Hyperparameter Tuning & Finalization**: Experiment with various models, tune hyperparameters, apply ensembles (e.g., Random Forest, GBDT), and finalize the model.
- **Prediction**: Predict on test/real-time data and assess performance using appropriate metrics.
- **Deployment**: Deploy the fully functioning prototype.
- **Scaling Up**: Add features like deep learning (sequence models) and NLP (LSTMs on scraped news data) to expand the project's scope.
