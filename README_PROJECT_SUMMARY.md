# Project Summary

## Objective
The objective of this project is to create a real-time stock market dashboard.

## Key Tasks & Features
- **Data Acquisition**: Fetch financial data from sources like Yahoo Finance API and Stocktwit API.
- **Exploratory Data Analysis (EDA)**: Understand features using statistical methods (moving averages, hypothesis testing, confidence intervals).
- **Data Visualization**: Generate box plots for price spreads, histograms/density plots for volatility and moving averages, and density plots for features like price-to-earnings ratio.
- **Web Dashboard**: Develop and deploy the base web application using Streamlit and Heroku to display insights.
- **Machine Learning Integration**:
  - Formulate ML problems to solve.
  - Preprocess data to handle volatility, outliers, missing data, imbalance, and scaling.
  - Benchmark simple linear models, engineer features, and select models.
  - Tune hyperparameters, validate, and use ensembles (Random Forest, GBDT).
  - Test predictions and evaluate performance.
- **Deployment & Scaling**: Deploy the prototype model and scale the project by adding deep learning models for sequence prediction and Natural Language Inference (NLI) on real-time stock news.
