# Project Objective
The main objective of this project is to create a real-time stock market dashboard.

## Overview
This repository contains a Streamlit-based multi-page application for analyzing and predicting stock market trends. The application fetches data from Yahoo Finance API, performs exploratory data analysis (EDA), visualizes the data using interactive charts, and employs machine learning techniques for prediction.

## Task Checklist
- **Data Acquisition**: Fetch financial data from Yahoo Finance API and Stocktwits API.
- **EDA**: Analyze features using moving averages, hypothesis tests, and confidence intervals.
- **Data Visualization**: Display box plots (spread of price), histograms/density plots (volatility), and other metrics (e.g., price-to-earning ratio).
- **Web Dashboard**: Deploy a base web application displaying the results using Streamlit.
- **Machine Learning**:
    - Formulate predictive problems.
    - Data Preprocessing (handle volatility, outliers, missing data, imbalance, scaling).
    - Model Selection & Feature Engineering (start with simple linear models, then tune/experiment).
    - Hyperparameter Tuning (experiment with ensembles like Bagging, Boosting, Stacking).
    - Test Data Prediction (evaluate model performance).
- **Prototype Deployment**: Deploy the fully functioning prototype.
- **Scaling Up**: Expand with deep learning (Sequence models/LSTM) and NLP inference from web-scraped news data.
