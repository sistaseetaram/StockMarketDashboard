# Project Summary

## Objective
The objective of this project is to create a real-time stock market dashboard.

## High-Level Tasks
- **Data Acquisition:** Gather data from Yahoo Finance and Stocktwit APIs.
- **Exploratory Data Analysis (EDA):** Analyze features using statistical methods like moving average, hypothesis tests, and confidence intervals.
- **Data Visualization:** Create box plots, histograms, and density plots to show stock price spreads, moving averages, volatility, and features like price-to-earning ratios.
- **Web Dashboard:** Build a base web application to display insights using Streamlit and deploy it to Heroku.
- **Machine Learning Formulation:** Define specific ML tasks or problems to solve.
- **Preprocessing:** Handle data volatility, outliers, missing data, imbalance, and scaling.
- **Model Selection & Feature Engineering:** Start with simple linear benchmarks, check feature importance, and experiment with new features.
- **Hyperparameter Tuning & Validation:** Experiment with different models (e.g., Random Forest, GBDT ensembles), tune hyperparameters, and finalize the model.
- **Prediction & Performance Evaluation:** Predict outcomes on test and real-time data and evaluate using appropriate metrics.
- **Prototype Deployment:** Deploy the fully functioning prototype model.
- **Scaling Up:** Expand functionality by developing deep learning sequence models and applying Natural Language Processing (e.g., LSTMs) to web-scraped stock news.
