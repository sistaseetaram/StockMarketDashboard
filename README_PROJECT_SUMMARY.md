# Project Summary: Stock Market Dashboard

## Objective
The primary objective of this project is to create a real-time stock market dashboard.

## Key Tasks and Milestones

1. **Data Acquisition**: Fetch data using the Yahoo Finance API and Stocktwit API.
2. **Exploratory Data Analysis (EDA)**: Understand features and data using statistical methods (moving averages, hypothesis tests, confidence intervals).
3. **Data Visualization**: Create box plots (spread of prices), histograms/density plots (volatility), and density plots for features like price-to-earning ratio.
4. **Initial Web Dashboard**: Build a base web application to display insights using Streamlit and deploy it on Heroku.
5. **Machine Learning Formulation**: Define ML problems/tasks to solve.
6. **Preprocessing**: Handle data volatility, outliers, missing data, imbalance, and scaling.
7. **Model Selection & Feature Engineering**: Start with simple linear models as benchmarks, analyze feature importance, experiment, and finalize features.
8. **Hyperparameter Tuning & Validation**: Experiment with different models and ensembles (Bagging, Boosting, Stacking like RF, GBDT), then finalize the model.
9. **Testing & Performance Check**: Predict results on test and real-time data, and evaluate using appropriate metrics.
10. **Deployment**: Deploy a fully functioning prototype model.
11. **Scale-up**: Add more features and data, such as developing a sequence model (Deep Learning) or performing Natural Language Inference on scraped stock news using LSTM and other state-of-the-art techniques.
