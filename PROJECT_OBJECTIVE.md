# Project Objective Summary

The primary objective of this project is to build a **Real-Time Stock Market Dashboard**. It encompasses a full end-to-end machine learning and data science pipeline, beginning with data acquisition and concluding with the deployment of predictive models.

## Key Tasks & Roadmap

1. **Data Acquisition**: Gather financial data using the Yahoo Finance API and Stocktwit API.
2. **Exploratory Data Analysis (EDA)**: Understand the features using statistical methods such as moving averages, hypothesis testing, and confidence intervals.
3. **Data Visualization**: Create visual insights like box plots (price spreads), histograms/density plots (volatility), and feature density charts (e.g., price-to-earnings ratios).
4. **Dashboard Creation**: Develop the base web application utilizing Streamlit and deploy the initial version to Heroku.
5. **Machine Learning Problem Formulation**: Define the specific ML tasks to solve using the stock data.
6. **Data Preprocessing**: Handle data volatility, outliers, missing values, class imbalance, and apply data scaling.
7. **Model Selection & Feature Engineering**: Establish baselines using simple linear models, analyze feature importance, and iteratively refine the feature set.
8. **Hyperparameter Tuning & Validation**: Experiment with different models (e.g., Random Forests, GBDTs) and ensemble techniques (Bagging, Boosting, Stacking), tuning them to finalize the best model.
9. **Prediction & Performance Check**: Evaluate model performance on test and real-time data using appropriate metrics.
10. **Prototype Deployment**: Deploy the fully functioning predictive model prototype.
11. **Scaling Up**: Extend the project by incorporating Deep Learning (sequence models) and Natural Language Processing (NLP) for drawing inferences from real-time web-scraped financial news.
